# Social Media Domains Reference List

A hosts-style domain blocklist for common social media and messaging platforms.

Use this list with Pi-hole or other network-level DNS filtering tools to help block major social media traffic.

## Raw list URL
https://raw.githubusercontent.com/Lacrimosica/social-media-domains-reference-list/main/list.txt

## Usage
1. Open Pi-hole Admin and go to Group Management > Adlists.
2. Paste the raw list URL into the Address field.
3. Click Add.
4. Go to Tools > Update Gravity and click Update.
